NuHash Python module
====================

Requires Python 3.11 or later, Python 2.x is *not* supported.


Instructions
------------

1. Install the Cython package:
   
   $ pip3 install Cython

2. Build the module:
   
   $ python3 setup.py build_ext

3. Run unit tests:
   
   $ python3 test_nuhash.py
